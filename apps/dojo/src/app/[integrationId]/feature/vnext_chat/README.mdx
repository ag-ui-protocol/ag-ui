# 🔮 VNext Chat

## What This Demo Shows

This demo showcases a streamlined chat experience built with the CopilotKit
vNext stack:

1. **Modern vNext UI**: Lightweight chat interface with streaming responses
2. **Integration-specific backend**: Uses the selected integration runtime
3. **Threaded conversations**: Messages remain scoped to a stable thread id

## How to Interact

This is a straightforward chat interface. Simply:

- Type your message and press Enter
- The selected integration agent responds in real time
- Conversations are threaded and maintain context

## ✨ Under the Hood

**What's happening technically:**

- The frontend connects to `/api/copilotkitnext/{integrationId}`
- The integration id determines which backend runtime handles requests
- Responses stream back in real-time

**Why this feature is useful:**

- **Consistent UX**: Same chat UI across different integrations
- **Fast experimentation**: Compare runtimes without changing frontend code
- **Simple surface area**: Minimal setup for end-to-end conversational testing
