# A2UI Advanced

## What This Demo Shows

Advanced A2UI patterns with frontend action handlers for instant optimistic UI responses.

1. **Frontend action handlers**: Uses `useA2UIActionHandler` for instant optimistic UI responses when buttons are clicked — no server round-trip needed
2. **Fallback chain**: Handler checks for agent-declared ops first, then builds custom confirmation ops on the frontend
3. **Same dynamic backend**: Reuses the dynamic schema agent; the advanced features are frontend-only
4. **Compare**: Click a "Select" button here vs. in the "A2UI Dynamic Schema" demo (which has no action handler) to see the difference
