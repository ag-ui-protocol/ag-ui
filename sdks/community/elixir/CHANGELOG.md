# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-01-28

### Added
- Core protocol types, events, and JSON encoding/decoding
- SSE transport parser and HTTP client
- Event normalization and compaction helpers
- Session reducer with JSON Patch support
- LiveView renderer, runner, and component kit (optional)
- Middleware and subscriber system
- Verification layer for event sequencing
- Demo LiveView app with mock agent SSE endpoint
